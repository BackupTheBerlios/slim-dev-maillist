Dear developers,

Before anything else, it is much pleasant to see that SLiM is back
into active development. I am not sure whether this problem was
previously reported, so here I go.

SLiM does not honour the X.Org keyboard layout configuration and
defaults to en_US. The problem arises especially when the physical
keyboard is (radically) different from the American English one (i.e.
French, Russian, etc.). This would mostly be a stopper to installing
SLiM, say, in French universities where users would probably be
un-aware of such a problem.

It would be nice if SLiM honoured the X.Org configuration and also
provided some shortcut (say, the "usual" <alt>+shift) to alternate the
keyboard layouts within SLiM. Alternatively, the necessary
configuration could be specified in slim.conf.

Thank you in advance,
Liviu
