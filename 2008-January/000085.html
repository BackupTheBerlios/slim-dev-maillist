Summary: switch user (feature request)

Details: I just installed the Fluxbuntu Linux distribution, which uses SLiM, and was disappointed with the lack of the "switch user" feature.  (That is, keep the current session running in background, and open a new login prompt for another user.)

I'll switch to GDM for now, because this feature is a must to me, but it would be nice to have it in SLiM.
