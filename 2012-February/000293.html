Summary: Default slim.conf should list icewm-session, not icewm

Details: The default slim.conf file currently lists "icewm" as an available session starter.  The proper command for starting Icewm is "icewm-session".

Starting with "icewm" leaves out (at least) the background setter, which messes up if SLiM tries to modify the root window (such as when pressing F1).
