Summary: slim-1.3.2 doesn't work with ConsoleKit: pam_ck_connector.so

Details: Because of this line in app.cpp:

pam.set_item(PAM::Authenticator::Host, "localhost");

It makes pam_ck_connector.so PAM module from ConsoleKit[1] think it's a remote host named "localhost", and not a real localhost and therefore renders ConsoleKit unusable with slim.

[1] http://www.freedesktop.org/wiki/Software/ConsoleKit

There's a downstream bug for Gentoo here:

http://bugs.gentoo.org/show_bug.cgi?id=346037

And a patch here:

http://bugs.gentoo.org/attachment.cgi?id=254799

And a long thread detailing the issue with ConsoleKit and slim here:

http://blog.flameeyes.eu/2010/11/11/draft-article-5307
