Summary: Support -nodaemon argument

Details: It would be nice if the "-nodaemon" argument would be supported.
If I understand it correctly, it doesn't have to do anything (default).

But the default inittab script appends this parameter automatically,
so by not accepting it slim refuse to start up when set as the DM...

Filtered it out in a shell wrapper meanwhile:
if [ "x$1" = "x-nodaemon" ]; then
  shift
fi
exec slim "$@"
